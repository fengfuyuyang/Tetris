# 俄罗斯方块小游戏

本项目是一个基于 SFML 库的俄罗斯方块游戏。游戏实现了经典的俄罗斯方块玩法，包括方块的生成、移动、旋转、下落以及消除行等功能

操作说明：

- 按下左方向键,左移方块
- 按下右方向键,右移方块
- 按下下方向键,下移方块
- 按下上方向键,旋转方块
- 按下空格键,立刻下落方块
- 按下加号键,增加方块下落速度
- 按下减号键,减小方块下落速度
- 按下P键,暂停或继续游戏
- 按下R键,重新开始游戏
- 按下Esc键,退出游戏
